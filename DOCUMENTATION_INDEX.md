3. **Data Export to CSV** - One-click download of anomaly data

- `GET /anomalies` - Returns anomaly history

## 💡 Pro Tips

1. **Change polling interval**: Edit `setInterval(..., 1000)` in App.tsx
2. **Change data retention**: Edit `1000` in app.state.data_points logic
3. **Change anomaly retention**: Edit `500` in app.state.anomaly_history logic
4. **Customize colors**: Modify Tailwind classes in UI components
5. **Add more statistics**: Use Python's statistics module
6. **Custom CSV fields**: Edit csv.writer.writerow() in main.py
